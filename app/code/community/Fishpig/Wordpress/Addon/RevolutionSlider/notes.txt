1.1.0.16
- Fixed cookie notice issue.

 1.1.0.0
- You can now use the Revolution Slider shortcodes anywhere on your site. This update requires Magento WordPress Integration version 4.2.0.0 or higher. To use the shortcode, use <?php echo Mage::helper('wordpress/filter')->process('[revslider alias="my-slider"]') ?> You will need to change the actual shortcode to match your shortcode.

- Added fix for https://github.com/hilios/jQuery.countdown/issues/41

1.0.0.1
- Bug fixes and improvements.

1.0.0.0
- Initial stable release of the Revolution Slider. This extension allows you to use the Revolution Slider plugin in the WordPress Admin and have the sliders display on your Magento WordPress integrated blog. This extension requires that you also install the Revolution Slider plugin in WordPress. You should also ensure that you have the latest version of Magento WordPress Integration installed.
